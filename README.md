
# Pera Konser Bilet Platformu

Bu proje Node.js (Express) ve MySQL kullanılarak geliştirilmiş bir full-stack web uygulamasıdır.

## Kurulum ve Çalıştırma Adımları

1. Bilgisayarınızda **Node.js** yüklü olduğundan emin olun.
2. **XAMPP** Kontrol Panelini açarak **Apache** ve **MySQL** servislerini başlatın (MySQL portunu 3307 olarak ayarlayın).
3. Proje ana dizininde bir terminal açarak gerekli paketleri yükleyin:
   ```bash
   npm install
Projede bulunan veritabanı.sql dosyasındaki kodları MySQL Workbench veya phpMyAdmin üzerinde çalıştırarak veritabanı tablolarını oluşturun.

Sunucuyu başlatmak için terminale şu komutu yazın:

Bash
node server.js
Tarayıcınızdan http://localhost:3000 adresine giderek projeyi görüntüleyebilirsiniz.
